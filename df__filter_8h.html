This function creates filtered dataframes of MC datas and Run datas with a snapshot function with columns "Muon_pt" , "Muon_mass" , "Muon_phi" , "Muon_eta". The filter is based on the following columns: "Muon_pt" , "Muon_eta", "nMuon", "Muon_dxy", "Muon_charge", "Muon_pfRelIso03_chg".

Parameters
filepath_MCdataframe of the MC datas
filepath_datasdataframe of the Run datas
filepath_MC_filfile path of the dataframe of the MC datas filtered
filepath_datas_fildataframe of the dataframe of the Run datas filtered
Returns
0: filtered dataframes have been created, 1: Doesn't exist the path of the files, 2: files with few columns, 3: files with few datas
